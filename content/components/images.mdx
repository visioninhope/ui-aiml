## Prerequisites

Before you start, ensure you have the following:

<Steps>
  <Step>
    Install the Required Dependencies in your project:
      - lucid react for icons
      - framer motion
      ```npm
      npm framer-motion lucid-react
      ```
  </Step>

  <Step>
    Create a new Project on Unsplash and obtain your API keys. For reference, check [Unsplash Documentation](https://unsplash.com/documentation#public-authentication).
    Watch the video tutorial below:
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">
        <AccordionTrigger>How to Get the API Keys - Video Tutorial</AccordionTrigger>
        <AccordionContent>
        <Steps>
            <Step>
             <h4>Step 1: Sign up on Unsplash and create a new application</h4>
              <VideoEmbed src="https://youtu.be/KJOGmNiuGUY?si=dMYd7fV-gu4Mk0HD"  
              text="
                  ## Create a New Application
                    To access Unsplash's API, you need to create a new application on their platform.
                    1. Go to the [Unsplash Developer Dashboard](https://unsplash.com/oauth/applications).
                    2. Click on **New Application** and fill in the required details.
                    3. Agree to the terms and click **Create Application**.
                    4. Once your application is created, you'll find your `Access Key` and `Secret Key` on the application page.
                    "/>
            </Step>
            <Step>
              <h4>Step 2: Get your Unsplash API keys</h4>
              <VideoEmbed src="https://youtu.be/KJOGmNiuGUY?si=Gvwj3fWlhcxfj4r2&t=14"  
                text='
                    ## API Keys

                      After creating your application, Unsplash will provide you with an API URL, `Access Key` required for making API requests.

                      1. Go to the [Unsplash Developer Dashboard](https://unsplash.com/oauth/applications).
                      2. Click on your application to view your `Access Key`.

                      Use these keys in your application to authenticate your API requests.

                      ### Environment Variables

                      ```env
                        NEXT_PUBLIC_UNSPLASH_ACCESS_KEY=REPLACE_WITH_YOUR_ACCESS_KEY
                      ```
                      '/>
            </Step>
          </Steps>
        </AccordionContent>
      </AccordionItem>
    </Accordion>
    Get Your API keys `Access Key` from Unsplash.
      ```env
        NEXT_PUBLIC_UNSPLASH_ACCESS_KEY=REPLACE_WITH_YOUR_ACCESS_KEY
      ```
  </Step>
  <Step>You&apos;re almost done, just a few steps left.</Step>
</Steps>
