  ## Supabase setup

  - Create a new project on [Supabase](https://app.supabase.io/).
  - Create a new table named `testimonials` int the SQL editor with the following schema:
  ``` SQL
      CREATE TABLE testimonials (
      id SERIAL PRIMARY KEY,
      name VARCHAR(255) NOT NULL,
      feedback TEXT NOT NULL,
      rating INT CHECK (rating >= 0 AND rating <= 5) NOT NULL
  );

  ```
  </Step>
  
  <Step>
  ## .env file
  Create the following `.env` variables:

    ```env
    NEXT_PUBLIC_MAGIC_API_KEY=YOUR_MAGIC_API_KEY
    NEXT_PUBLIC_MAGIC_PROJECT_ID=YOUR_MAGIC_PROJECT_ID
    ```

    You can find the environment variables for your project in the API settings under project settings.
