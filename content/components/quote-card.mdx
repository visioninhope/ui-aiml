      ## props

      | Prop name         | Type                                      | Default   | Description                                                                 |
      | ----------------- | ----------------------------------------- | --------- | --------------------------------------------------------------------------- |
      | default_velocity  | number                                    | 0.5       | The default velocity for the marquee effect.                                |
      | className         | string                                    | `font-display text-center text-9xl font-bold tracking-[-0.02em] text-black drop-shadow-sm dark:text-white md:text-7xl md:leading-[5rem]` | CSS class names to apply to the component.                                   |
      | gradientColors    | `{ light: string[], dark: string[] }`       | `{ light: ["#000000", "#000000"], dark: ["#ffffff", "#ffffff"] }` | Colors for the gradient effect in light and dark modes.                     |
      | gradientType      | "linear" \| "radial"                      | `linear`  | Type of gradient to use (linear or radial).                                 |
      | parallax          | boolean                                   | true      | Whether to enable the parallax effect.                                      |
      | numQuotes         | number                                    | 3         | Number of quotes to fetch and display.                                      |
      | marqueeType       | `"marqueeleft" \| "marqueeright" \| "scrollmarquee" \| "nomarquee"` | `"scrollmarquee"`| Type of marquee effect to use.                                              |
      | updateInterval    | number                                    | 10        | Interval in seconds to fetch new quotes.                                    |
      | font              | `{ family: string; style: string }`         | `{ family: "Roboto", style: "normal" }` | Font family and style for the text.                                         |
      | textSize          | string                                    | `text-9xl` | CSS class for adjusting the text size.                                      |
